# ChickRubGo - 智能搜索引擎

ChickRubGo 是一款现代化的智能搜索引擎界面，集成了美观的壁纸展示功能，旨在为用户提供高效便捷的搜索体验。

## 功能特性

- **智能搜索**：支持关键词搜索，一键跳转至Bing搜索结果
- **精美壁纸**：自动获取高质量全风格壁纸作为背景
- **个性化设置**：可控制壁纸显示/隐藏
- **自定义搜索引擎**：可在设置中选择必应、百度或自定义搜索引擎
- **响应式设计**：适配各种屏幕尺寸，提供良好的移动设备体验

## 默认配置
- 壁纸模块：默认显示

## 使用方法

1. 输入关键词到搜索框中
2. 按回车键或点击"搜索"按钮进行搜索
3. 点击右上角齿轮图标进入设置面板，可自定义显示选项
4. 点击刷新按钮可更换背景壁纸

## 快捷标签

页面提供了多个热门搜索标签，包括：
- 前端开发
- AI工具
- 设计资源
- 学习教程
- 影视推荐

## 设置说明

- **显示壁纸**：控制背景壁纸的显示与隐藏

所有设置都会自动保存到本地存储中，下次访问时会保持之前的设置。

## 浏览器兼容性

- Chrome, Firefox, Safari, Edge (最新版本)
- 支持移动端浏览器
