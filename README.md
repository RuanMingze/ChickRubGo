# ChickRubGo - 智能搜索引擎

ChickRubGo 是一款现代化的智能搜索引擎界面，集成了丰富的小组件功能和个性化设置，旨在为用户提供高效便捷的搜索体验。

## 功能特性

- **智能搜索**：支持关键词搜索，可选择必应、百度或自定义搜索引擎
- **精美壁纸**：自动获取高质量全风格壁纸作为背景
- **小组件系统**：集成多种实用小组件
  - 记事本（支持Markdown格式，可导入/导出文件）
  - 天气（支持自动定位和手动城市设置）
  - 木鱼（点击积累功德值）
  - 闹钟（支持多闹钟设置和番茄钟功能）
  - 计算器（基础计算功能）
  - 每日一言（随机获取名言警句）
  - 今天吃什么（随机选择食物）
- **模式切换**：支持简洁模式和丰富模式
- **深色模式**：可切换深色/浅色主题
- **背景音乐**：支持自定义背景音乐，可上传音频文件或MP4视频文件
- **彩蛋功能**：输入特定代码触发彩蛋模式
- **响应式设计**：适配各种屏幕尺寸，提供良好的移动设备体验

## 使用方法

1. 输入关键词到搜索框中，按回车键或点击"搜索"按钮进行搜索
2. 点击右上角齿轮图标进入设置面板，可自定义各种显示选项
3. 点击刷新按钮可更换背景壁纸
4. 点击小组件图标打开对应功能窗口
5. 点击"编辑"按钮可进入小组件编辑模式，可添加/删除/排序小组件

## 设置说明

- **显示壁纸**：控制背景壁纸的显示与隐藏
- **显示时间**：控制时间和日期的显示
- **深色模式**：切换深色/浅色主题
- **显示淡入动画**：控制页面元素的淡入动画效果
- **搜索引擎**：选择必应、百度或自定义搜索引擎
- **OpenWeatherMap API Key**：设置天气小组件的API密钥
- **自动获取定位**：控制天气小组件是否自动获取定位
- **城市**：手动设置天气小组件的城市
- **背景音乐**：上传自定义背景音乐文件

所有设置都会自动保存到本地存储中，下次访问时会保持之前的设置。
