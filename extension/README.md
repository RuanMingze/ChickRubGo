# ChickRubGo浏览器扩展

这是一个将新标签页替换为ChickRubGo智能搜索引擎的浏览器扩展。

## 功能特性

- 将新标签页替换为ChickRubGo搜索引擎
- 保持所有原有的搜索功能
- 自动加载壁纸和新闻（根据设置）
- 支持个性化设置（壁纸显示、新闻显示、新闻数量等）

## 安装方法

### Edge浏览器

1. 打开Edge浏览器，访问 `edge://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载解包"按钮
4. 选择此扩展文件夹（ChickRubGo/extension）
5. 扩展安装完成

### Chrome浏览器

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"按钮
4. 选择此扩展文件夹（ChickRubGo/extension）
5. 扩展安装完成

## 使用说明

安装扩展后，每次打开新的标签页时，都将显示ChickRubGo搜索引擎界面，您可以：

- 在搜索框中输入关键词进行搜索
- 点击右上角齿轮图标进入设置面板
- 控制壁纸和新闻的显示/隐藏
- 调整新闻显示数量
- 点击刷新按钮更换壁纸

## 扩展文件结构

```
extension/
├── manifest.json     # 扩展配置文件
├── index.html        # 新标签页入口文件
├── favicon-2.png       # 扩展图标（复用项目图标）
└── README.md         # 说明文档
```

## 注意事项

- 扩展需要网络连接才能正常加载壁纸和新闻
- 部分功能可能因网络环境而有所差异
- 扩展会保存您的个性化设置到本地存储

## 开发

如需修改扩展行为，可以编辑 `index.html` 文件，调整对 ChickRubGo 页面的引用。
